# HostConnect

**Short-Term Rentals & Corporate Housing in Pittsburgh**

Landing page for HostConnect — a booking site for furnished short-term rentals and corporate housing in Pittsburgh, PA.

## Run locally

Open `index.html` in a browser, or use a local server:

```bash
# Python 3
python3 -m http.server 8000

# Node (npx)
npx serve .
```

Then visit `http://localhost:8000`.

## Structure

- **index.html** — Main page (hero, trust, how it works, search CTA, guarantees, footer)
- **styles.css** — Layout and styling (dark theme, gold accent)
- **script.js** — Header scroll state, mobile menu toggle
